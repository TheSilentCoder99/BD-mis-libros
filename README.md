# BD-mis-libros
Esta es una base de datos en la que registro mis libros leídos.

## Objetivo
Esta base de datos fue creada como proyecto de práctica para aprender y afianzar aspectos básicos de SQL como:

- Importación de datos desde CSV
- Consultas SQL
- Diferentes tipos de JOIN
- Uso de vistas en MySQL
  
## Datos
La base de datos fue construida a partir de archivos CSV
(autores, países, géneros y libros).

El archivo `Mis_libros.sql` contiene la base de datos final
con tablas, datos y vistas ya creadas.

El objetivo principal del proyecto es servir como base de datos sobre la que practicar consultas.
